# Git Quick Reference

A practical reference for the Git and GitHub concepts used while building TPM Copilot.

## Core Workflow

```text
Edit → Save → Review → Commit → Push
```

- **Edit:** Make changes to a project file.
- **Save:** Write those changes to the computer.
- **Review:** Inspect what changed before recording it.
- **Commit:** Create a named checkpoint in the local Git history.
- **Push:** Send local commits to GitHub.
