Vibe Engineering Lab

Purpose

Welcome to Vibe Engineering Lab—my personal engineering workspace for learning modern AI engineering, enterprise software architecture, and AI-assisted software development.

This repository documents my journey from Technical Program Manager to AI Engineer by focusing on the principles, tools, and practices used to build enterprise-grade AI solutions.

Rather than learning technology in isolation, every lesson and project in this repository contributes toward building a real-world application called TPM Copilot.

⸻

Vision

The long-term vision is to design and build TPM Copilot—an AI-powered Technical Program Management platform capable of helping enterprise program managers understand complex initiatives by connecting AI with systems such as:

* Jira
* Confluence
* GitHub
* Microsoft Teams
* Outlook
* SharePoint
* Additional enterprise tools

TPM Copilot will eventually assist with:

* Executive status reporting
* Risk and dependency analysis
* Architecture discovery
* Documentation search
* Program intelligence
* AI-assisted decision support

⸻

Learning Philosophy

This repository is based on a simple principle:

Learn by building.

Rather than completing disconnected tutorials, every lesson contributes directly toward solving real enterprise problems.

The emphasis is on understanding:

* Why technologies exist
* The problems they solve
* How they work together
* How they support enterprise architecture
